# NusantaraSpy - Pegasus Style untuk Termux Non-Root

Spyware canggih berjalan sepenuhnya di Termux tanpa akses root.  
Mengumpulkan: Kontak, SMS, Lokasi GPS, Kamera, Mikrofon, Info Perangkat, dan banyak lagi.
